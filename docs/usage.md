# Como usar o Port Scanner

1. Instale o Python 3 no seu computador.
2. Abra o terminal na pasta do projeto:
python src/port_scanner.py
3. Digite o IP ou domínio para iniciar o scan.

### Portas analisadas no scan padrão
- 21 (FTP)
- 22 (SSH)
- 23 (Telnet)
- 25 (SMTP)
- 53 (DNS)
- 80 (HTTP)
- 110 (POP3)
- 139 (NetBIOS)
- 443 (HTTPS)
- 445 (SMB)
- 3389 (RDP)

Este projeto é apenas para fins educacionais.
